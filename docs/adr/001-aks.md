# ADR-001: Use Azure Kubernetes Service (AKS)

**Status:** Accepted
**Date:** 2026-01-14

Need a Kubernetes platform for running team workloads. AKS was a pretty easy call since the org is already on Azure: auth, networking, billing all integrate natively. Control plane is free (only pay for nodes), Azure AD integration means I can skip bolting on a separate identity provider, and managed upgrades and node patching are included.

Tradeoff is vendor lock-in to Azure, but I'm already committed there. Some K8s features lag behind upstream releases but nothing that's blocked me yet.
